Curva de tendência dos algoritmos de ordenação

Análise da curva de tendência do desempenho de cada algoritmo de ordenação atuando sobre n elementos de um array em função do tempo.

Análise dos algoritmos


Nesta sessão, iremos analisar a curva de tendência de cada um dos algoritmos previamente estudados:
  • Bubble Sort
  • Selection Sort
  • Insertion Sort
  • Merge Sort

Podemos analisar, por meio do gráfico, que o método Merge Sort é o mais consistente e eficiente pois independente do tamanho do array informado, seu tempo para ordenar é praticamente o mesmo. Os métodos Insertion Sort e Selection Sort têm desempenho praticamente idêntico e ficam no meio do caminho, pois seus tempos de execução aumentam conforme o tamanho array aumenta, porém não de forma tão acentuada quanto o Bubble Sort, que após atingida uma certa quantidade de elementos no array, seu uso fica praticamente inviável em termos de custo operacional e tempo de execução.

Complexidade de cada algoritmo de ordenação

Com base na literatura, estas são as complexidades de cada algoritmo ao ordenar um array de n elementos usando a notação big O:

Algoritmo Melhor caso Caso médio Pior caso
Bubble Sort O(n²) O(n²) O(n²)
Selection Sort O(n²) O(n²) O(n²)
Insertion Sort O(n) O(n²) O(n²)
Merge Sort O(nlognn) O(nlogn) O(nlogn)
